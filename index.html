# DraggableJS

MAKE YOUR HTML ELEMENT DRAGGABLE WITH SIMPLE STEPS.

  1. ADD FDraggable.js file in your project directory.
  2. Create a separate file main.js (or with a preferred name) and include it in your project.
  3. import { FDraggable } from './FDraggable.js';
    document.addEventListener('DOMContentLoaded', () => {
     const elementId = 'myDraggableElement'; // myDraggableElement - DRAGGABLE ELEMENT ID
     let myDraggable = new FDraggable(elementId);
    });
Sample URL to run - http://localhost/DraggableJS/index.html
